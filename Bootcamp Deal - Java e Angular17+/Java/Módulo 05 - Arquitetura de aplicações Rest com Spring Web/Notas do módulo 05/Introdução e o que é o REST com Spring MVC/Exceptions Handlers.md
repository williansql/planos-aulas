## @ControllerAdvice

O **@ControllerAdvice** é uma anotação, para lidar com as exceções globalmente.

## @ExceptionHandler

O **@ExceptionHandler** é uma anotação usada para tratar as exceções específicas e enviar as respostas personalizadas ao cliente.
